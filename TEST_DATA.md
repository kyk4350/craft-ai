# ContentCraft AI - 테스트 가이드

## 🚀 빠른 시작

1. **백엔드 실행 확인**: http://localhost:8000/docs 접속 → Swagger UI 보이면 OK
2. **프론트엔드 접속**: http://localhost:5173
3. **아래 데이터 복사해서 입력**

---

## 📝 입력 폼 가이드

### 1️⃣ 제품 정보 섹션

**제품명** (필수)
```
글로우 비타민C 세럼
```

**제품 설명** (필수)
```
프랑스산 천연 비타민C 20% 함유, 피부 톤 개선 및 잡티 완화에 효과적인 고농축 세럼
```

**카테고리** (드롭다운 선택)
- 선택: `화장품/뷰티`

---

### 2️⃣ 타겟 고객 정보 섹션

**나이대** (드롭다운 선택)
- 선택: `20대`

**성별** (드롭다운 선택)
- 선택: `여성`

**관심사** (버튼 클릭으로 선택, 최소 1개 필수)
- 클릭: `뷰티`, `자기관리`, `쇼핑`
- (선택하면 파란색으로 변하고 위에 태그로 표시됨)

---

### 3️⃣ 카피 톤 선택

**카피 톤** (라디오 버튼 선택)
- 선택: `프로페셔널 (40-50자)`

---

### 4️⃣ 생성 버튼

**파란색 큰 버튼 클릭**: `콘텐츠 생성하기`

---

## ⏱️ 예상 소요 시간

- **전체 프로세스**: 약 30-40초
- **진행 순서**:
  1. 전략 생성 (10초)
  2. 카피 생성 (5초)
  3. 이미지 프롬프트 변환 (5초)
  4. 이미지 생성 (20-25초) ← 가장 오래 걸림

---

## ✅ 예상 결과

### 생성되는 것:
1. **3가지 마케팅 전략** (감성적, 이성적, 사회적)
2. **광고 카피** (프로페셔널 톤, 40-50자)
3. **해시태그** 3-5개
4. **AI 생성 이미지** (1024x1024 고품질)

### 할 수 있는 것:
- ✅ 전략 선택 (3개 중 클릭)
- ✅ 이미지 다운로드
- ✅ 카피 복사
- ✅ 새로운 콘텐츠 생성

## 🐛 문제 해결

### "콘텐츠 생성에 실패했습니다" 에러
- 백엔드가 실행 중인지 확인: http://localhost:8000/docs
- 환경변수 확인: `backend/.env` 파일에 API 키가 있는지 확인
  - `GEMINI_API_KEY`
  - `REPLICATE_API_TOKEN`

### 이미지가 안 보여요
- 로컬 URL 실패 시 자동으로 원본 URL로 전환됨
- 개발자 도구(F12) → Console 탭에서 에러 확인

### 로딩이 너무 오래 걸려요
- 이미지 생성은 20-25초 소요 (정상)
- 40초 이상 걸리면 네트워크 확인
